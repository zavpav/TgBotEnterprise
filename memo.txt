Сервисы
Все сервисы, кроме Главного, не думают. Получают данные извне, отправляют в RabbitMQ, или наоборот получают данные из RabbitMQ и выполняют запрос, отправляя ответ обратно в RabbitMQ.

1. Главный. Выполняет работу бота
    Обрабатывает входящие сообщения из телеграма
        Работа с настройками
        Получение статистики по версии
    Рассылает изменения из Redmine
        При назначении задачи пользователю 
        При изменении задачи, назначенной пользователю (если получится убрать просто редактирование или если автор не совпадает с текущим исполнителем)
    Рассылает изменения из Jenkins
        Определяет каким пользователям надо уведомление об этом статусе и в RabbitMQ закладывает нужыне сообщения для чтения телегой
    

2. Работа с телеграмом. 
    Обрабатывает входящие сообщения. Записывает их в RabbitMQ
    Отправляет из RabbitMQ сообщения в телеграм.

3. Вебморда-админка
    Отображает список пользователей.

4. Работа с git.
    По факту нужен только для функциональности "Есть Чё?"

5. Работа с Jenkins
    Получает информацию о сборках Jenkins.
    По таймеру дергает статистику Jenkins, если поменялась - пихает в RabbitMQ сообщение.
    Входящие сообщения:
       За какими джобами следим
    Основное исходящее сообщение 
      - Имя джобы
      - Дата сборки
      - Номер сборки
      - Статус сборки (запущена/завершена успешно/провалена)
      - Кто запустил
      - Какие коммиты попали

6. Работа с Redmine
    Входящие сообщения:
        За какими проектами следим
   
   Получить статистику по версии:
      Входящее сообщение:
        Проект Redmine
        Версия
      Исходящее сообщение:
        Список задач с информацией по задачам.
        - Общая информция по задаче

   Исходящее сообщение о смене статуса задачи:
      - Внутренний статус (сменен пользователь: новая или сменился пользователь, задача изменена: изменен текст-вложения-названия-версии)
      - Кто внес изменения (?)
      - Общая информция по задаче

    Общая информция по задаче в сообщении:
      - Имя проекта Remine (внутренниие имена проекта не учитываются)
      - Версия
      - Номер задачи
      - Текущий статус задачи Remine
      - Название задачи
      - Текст задачи
      - Резолюция по задаче
      - Дата изменения
      - Текущий исполнитель
